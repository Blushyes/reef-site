# reef-site

Landing page for [**reef**](https://github.com/Blushyes/reef) — the minimal dev terminal for the AI coding era.

Published via GitHub Pages: **https://blushyes.github.io/reef-site/**

## Stack

A single self-contained `index.html`. No build step, no dependencies beyond two Google Fonts.

The hero section is a live, keyboard-navigable mock of the Reef TUI — Files / Git / Graph tabs, diff viewer, commit graph. Everything is vanilla JS + CSS grid.

## Local preview

```sh
python3 -m http.server 3001
```

Then open http://127.0.0.1:3001.
